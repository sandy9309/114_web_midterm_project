# 114_web_midterm_project
網頁程式設計期中小組專案
--專案名稱
FollowOurStar - SEVENTEEN 十週年應援網站
--組員
412630039鄧詠妍、412630930劉姵岑
--專案簡介
我們做了一個SEVENTEEN十週年應援活動展示網站，為的是慶祝 SEVENTEEN 出道十週年，提供線上展覽預約、應援物購買登記及會員註冊/登入模擬功能。網站具備完整的表單驗證(輸入驗證、即時金額計算等)與流暢的使用者體驗。
--使用技術
HTML5、CSS3、Bootstrap 5、JavaScript 、LocalStorage
--功能特色
1. 完整表單驗證：包含必填、Email/手機格式、密碼強度及一致性驗證，並以輸入框旁文字提示取代彈跳視窗，優化使用者體驗。
2. 會員系統模擬：使用 localStorage 模擬會員註冊、登入驗證與登入狀態切換。
3. 即時互動功能：應援物總金額即時計算、會員密碼強度即時反饋。
4. 資料暫存：使用 localStorage 記住展覽預約表單內容，防止資料遺失。
5. 響應式設計 (RWD)：採用 Bootstrap 5 確保網頁在不同裝置上皆能良好顯示。
6. 視覺優化：實現平滑滾動、表單區塊 Hover 動畫等效果。
